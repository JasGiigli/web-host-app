HTML Tutorial

Introduction to HTML

HTML, which stands for HyperText Markup Language, is the standard markup language for creating web pages. It structures content on the internet by using a series of elements to define different parts of a page.

Basic Structure of an HTML Page

An HTML page consists of several key components:

  • DOCTYPE Declaration: Specifies the version of HTML being used.
  • HTML Element: The root element that encapsulates the entire HTML document.
  • Head Element: Contains meta-information about the document, such as its title, character set, and stylesheets.
  • Body Element: Contains the visible content of the web page.

HTML Elements and Tags

HTML uses elements and tags to define the structure of a web page. Elements consist of a start tag, content, and an end tag.

For example:

                
        <h1>This is a Heading</h1>
        <p>This is a paragraph.</p>

Common HTML Tags

Here are some commonly used HTML tags:

  • <h1> to <h6> for headings
  • <p> for paragraphs
  • <a> for links
  • <img> for images
  • <ul> and <li> for unordered lists
  • <ol> and <li> for ordered lists

Creating Your First HTML Page

To create a simple HTML page, use a text editor and save the file with a .html extension. For example: 

                
        <!DOCTYPE html>
        <html>
        <head>
            <title>My First HTML Page</title>
        </head>
        <body>
            <h1>Hello, World!</h1>
            <p>This is my first web page.</p>
        </body>
        </html>
